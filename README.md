## Deploy

```sh
git clone https://github.com/FayasNoushad/Calculator-Bot/tree/main
cd Calculator-Bot
pip3 install -r requirements.txt
# <Create Variables appropriately>
python3 main.py
```

## Variables

- `BOT_TOKEN` Your bot token from @BotFather
